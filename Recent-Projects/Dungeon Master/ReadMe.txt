How to operate the game:


To move around:
On keyboard: Use the WASD keys.

To shoot: 
On mouse: right click  on the right side of the screen.

To use power-up:
On keyboard: Use the Q key.

To use dodge :
On keyboard: Use the E key.

To use invincible:
On keyboard: Use the space key.

I have impletment xml file for all the equipment. The descriptions are in Chinese,
 so I have also used the autotranslator to translate the description and its attributes.
They are already set for Chinese text translate to English.
